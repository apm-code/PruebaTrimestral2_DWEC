# API Series – Angular 
## Prueba Trimestral 2Trimestre - Desarrollo Web Entorno Cliente (DWEC)

Aplicación SPA desarrollada en Angular que consume una API externa de series.

## Funcionalidades
- Listado de series (GET)
- Crear nueva serie mediante formulario (POST)
- Navegación con Router
- Header y Footer comunes en las páginas

## Tecnologías usadas
- Angular
- Router
- HttpClient

## API utilizada
https://peticiones.online/api/series

## Ejecución en local

Instalar dependencias:
```
npm install
```

Arrancar proyecto:
```
ng serve -o
```
